## <a name="parte4">4 - Instalando o PHPUnit</a>

```
composer init

```

```json
{
    "name": "josem/projeto_1_aula",
    "type": "project",
    "license": "MIT",
    "authors": [
        {
            "name": "josemalcher",

        }

    "require": {},
    "autoload": {
        "psr-4": {
            "App\\": "app/"
        }
    }
}

```

```
composer update
```

- Instalar o PHPUnit

```
composer require --dev phpunit/phpunit ^7
```
